# 🛒 E-Commerce Sales Data Analysis

## 📌 Project Overview
This project focuses on cleaning and analyzing a real-world e-commerce dataset to extract meaningful business insights.

The dataset was sourced from Kaggle (Superstore Sales Dataset) and contains detailed information about orders, customers, products, sales, and profit.

---

## 🎯 Objective
- Clean messy real-world data  
- Perform data analysis  
- Generate business insights  
- Visualize key trends  

---

## 📂 Dataset
- Source: Kaggle  
- Dataset: Superstore Sales Dataset  
- Features include:
  - Order details  
  - Customer information  
  - Product categories  
  - Sales and profit  

---

## 🔧 Tools & Technologies
- Python  
- Pandas  
- Matplotlib  
- Jupyter Notebook  

---

## 🧹 Data Cleaning Steps
- Renamed columns for better readability  
- Converted date columns to datetime format  
- Handled missing values  
- Removed duplicate records  

---

## ⚙️ Feature Engineering
- Extracted:
  - Order year  
  - Order month  
- Created:
  - Sales per quantity  

---

## 📊 Data Analysis
- Total sales calculation  
- Category-wise sales analysis  
- Region-wise performance analysis  

---

## 📈 Visualizations
- 📉 Yearly Sales Trend (Line Chart)  
- 📊 Sales by Category (Bar Chart)  
- 🥧 Sales by Region (Pie Chart)  

---

## 💡 Key Insights
- Certain product categories generate higher revenue  
- Sales vary significantly across regions  
- Trends over time help understand business growth
